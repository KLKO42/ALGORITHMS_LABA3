## Преимущество стека выбрано по следующим причинам:

1. Скобки работают по правилу LIFO: последняя открытая должна закрыться первой. Стек идеально подходит для данной задачи.
2. Легко сравнить закрывающую скобку с вершиной стека за O(1), не нужно перебирать все элементы.
3. Открывающие скобки "запоминаются" в стеке, а при встрече закрывающей — "вспоминается" последняя открытая. И как раз стеком удлбнее.
